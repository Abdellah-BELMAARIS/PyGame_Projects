CLI Terminal Setup

Want to run them natively as desktop apps?

This entire collection is structured as a professional, multi-project Python workspace. You can run all 15 games locally using standard python CLI commands.

  • 1 Uses standard Python event pooling for extremely smooth desktop rendering.
  • 2 Spawns isolated, lightweight OS processes (`subprocess.Popen`) from the central menu launcher.
  • 3 Zero configuration needed: dependency packages are automatically resolved.

Quickstart Desktop Commands

PowerShell / Bash
1. Clone Repository
git clone https://github.com/Abdellah-BELMAARIS/PyGame_Projects.git
2. Enter and Install PyGame
cd PyGame_Projects ; pip install pygame
3. Start Arcade Launcher
python main.py
